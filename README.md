# ЁЯУК Exploratory Data Analysis (EDA) on Books Dataset

## ЁЯУМ Project Overview

This project was completed as part of the **CodeAlpha Data Analytics Internship (Task 2: Exploratory Data Analysis)**.

The dataset was created by scraping book information from the **Books to Scrape** website during Task 1. Exploratory Data Analysis (EDA) was then performed to understand the dataset, identify patterns, clean the data, and generate meaningful visualizations.

---

## ЁЯОп Objectives

* Explore the dataset structure
* Check data types and dataset dimensions
* Identify missing values
* Detect duplicate records
* Clean and preprocess the data
* Perform statistical analysis
* Visualize the data using charts
* Extract meaningful insights

---

## ЁЯУВ Dataset

The dataset contains information about books collected from the **Books to Scrape** website.

### Dataset Columns

| Column | Description       |
| ------ | ----------------- |
| Title  | Name of the book  |
| Price  | Price of the book |
| Rating | Book rating       |
