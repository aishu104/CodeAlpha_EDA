# 📸 Project Screenshots

This folder contains the visualizations generated during the Exploratory Data Analysis (EDA) of the **Books Dataset**.

## Included Images

### 📊 Price Distribution

**File:** `price_distribution.png`

* Displays the distribution of book prices.
* Helps identify common price ranges and the overall spread of prices.

---

### ⭐ Rating Distribution

**File:** `rating_distribution.png`

* Shows the number of books in each rating category.
* Helps understand which ratings are most common in the dataset.

---

### 📈 Price vs Rating

**File:** `price_vs_rating.png`

* Scatter plot illustrating the relationship between book prices and ratings.
* Useful for observing whether higher-rated books tend to have higher prices.

---

## Purpose

These visualizations were generated using **Matplotlib** and **Seaborn** as part of the **CodeAlpha Data Analytics Internship - Task 2 (Exploratory Data Analysis)**. They help summarize the dataset, reveal patterns, and support data-driven insights.
